# QR Contact

Application web légère de carte de visite numérique permettant de générer un QR Code contenant les informations de contact d'une personne afin de faciliter leur ajout dans le répertoire d'un smartphone.

## Fonctionnalités

* Génération instantanée d'une fiche contact
* Création d'un QR Code à partir des informations saisies
* Compatible mobile et ordinateur
* Installation possible comme application (PWA)
* Fonctionnement sans serveur

## Cas d'usage

- Cartes de visite numériques
- Réseautage lors d'événements professionnels
- Salons et conférences
- Commerciaux et indépendants
- Partage rapide de coordonnées sans support papier

## Confidentialité et protection des données

Cette application fonctionne entièrement sur votre appareil.

* Aucune donnée personnelle n'est transmise à un serveur.
* Aucune information n'est stockée dans une base de données.
* Aucun compte utilisateur n'est nécessaire.
* Les informations saisies restent uniquement sur l'appareil utilisé (ordinateur ou smartphone).
* Le QR Code est généré localement dans le navigateur.

L'utilisateur conserve ainsi un contrôle total sur ses données personnelles.

## Technologies utilisées

* HTML5
* JavaScript
* Progressive Web App (PWA)
* Service Worker pour le fonctionnement hors ligne

## Installation locale

Cloner le dépôt :

```bash
git clone https://github.com/annece29-netizen/qr-contact.git
```

Ouvrir ensuite le fichier `index.html` dans un navigateur.

## Déploiement

L'application peut être déployée sur :

* GitHub Pages
* Netlify
* Cloudflare Pages
* Tout hébergement statique compatible HTML/CSS/JavaScript

## Utilisation

1. Saisir les informations du contact.
2. Générer le QR Code.
3. Scanner le QR Code avec un smartphone.
4. Enregistrer le contact dans le carnet d'adresses.
